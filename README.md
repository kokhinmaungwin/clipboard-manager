# Clipboard Manager

A simple and powerful clipboard management Progressive Web App (PWA) built with React and Vite.

---

## Features

- Manage your clipboard history easily  
- PWA support: Install on your device  
- Offline support with service worker  
- Lightweight and fast

---

## Installation

Clone the repository:

```bash
git clone https://github.com/kokhinmaungwin/clipboard-manager.git
```
```bash
cd clipboard-manager
```
Install dependencies:
```bash
npm install
```
Run development server:
```bash
npm run dev
```
Build for production:
```bash
npm run build
```
Deploy to GitHub Pages:
```bash
npm run deploy
```

---

## Usage

Open the app in your browser or install it as a PWA

Copy text and it will be saved in clipboard history

Manage, delete, and use saved clipboard entries
